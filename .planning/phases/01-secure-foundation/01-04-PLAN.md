**lib/domain/use_cases/profile_use_case.dart:**
Implement `ProfileUseCase` with constructor injection:
```dart
class ProfileUseCase {
  final ProfileRepository _repo;
  final EntitlementTier _tier;
  static const int freeTierLimit = 2;

  ProfileUseCase(this._repo, this._tier);

  Future<FamilyProfile> createProfile(ProfileCreateRequest request) async { ... }
  Future<FamilyProfile> updateProfile(String id, ProfileUpdateRequest request) async { ... }
  Future<void> deleteProfile(String id) async { ... }
}
```

`createProfile()`:
1. If `_tier == EntitlementTier.free`, call `_repo.countActive()`
2. If count >= freeTierLimit, throw `ProfileLimitReached()` (a plain Dart exception class)
3. Otherwise, build a `FamilyProfile` with: `id = const Uuid().v4()`, `createdAt = DateTime.now().toUtc()`,
   `updatedAt = DateTime.now().toUtc()`, `synchronized = false`, all fields from request
4. Call `_repo.upsert(profile)` and return the profile

`updateProfile()`:
1. Fetch existing profile via `_repo.getById(id)` — throw `ProfileNotFound()` if null
2. Use `.copyWith(...)` to apply request fields + `updatedAt = DateTime.now().toUtc()` + `synchronized = false`
3. Call `_repo.upsert(updatedProfile)` and return it

`deleteProfile()`: calls `_repo.softDelete(id)` only — no hard delete, no cascade
(cascade handled by DB FK constraint or later by custom field soft-delete in use case).

Define exception classes in the same file:
```dart
class ProfileLimitReached implements Exception { const ProfileLimitReached(); }
class ProfileNotFound implements Exception { const ProfileNotFound(this.id); final String id; }
```

```dart
// Profile list stream — UI subscribes to this for reactive updates
@riverpod
Stream<List<FamilyProfile>> profileList(Ref ref) async* {
  final repo = await ref.watch(profileRepositoryProvider.future);
  yield* repo.watchActive();
}

// Use case providers — UI calls these for mutations
@Riverpod(keepAlive: true)
Future<ProfileUseCase> profileUseCase(Ref ref) async {
  final repo = await ref.watch(profileRepositoryProvider.future);
  final tier = ref.watch(entitlementTierProvider);
  return ProfileUseCase(repo, tier);
}

@Riverpod(keepAlive: true)
Future<CustomFieldUseCase> customFieldUseCase(Ref ref) async {
  final repo = await ref.watch(customFieldRepositoryProvider.future);
  return CustomFieldUseCase(repo);
}
```
